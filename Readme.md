**[marcosdurval/fullcycle](https://hub.docker.com/r/marcosdurval/fullcycle)**

## Como usar

### Executar a aplicação

```bash
docker run marcosdurval/fullcycle
```

### Build local

```bash
docker build -t fullcycle .
```
